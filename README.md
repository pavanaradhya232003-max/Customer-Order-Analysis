# 📊 Customer Sales Analytics Using Python and Power BI

## 📌 Project Overview

This project performs end-to-end analysis of customer and sales data using **Python** and **Power BI**. The objective is to transform raw sales data into meaningful business insights through data cleaning, exploratory data analysis (EDA), visualization, and interactive dashboards.

The project helps businesses understand customer behavior, sales trends, product performance, and overall business performance for better decision-making.

---

## 🎯 Objectives

- Clean and preprocess customer and order datasets.
- Perform exploratory data analysis (EDA).
- Analyze customer purchasing behavior.
- Identify sales trends and revenue patterns.
- Create interactive dashboards in Power BI.
- Generate actionable business insights.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/Customer-Sales-Analytics-Using-Python-and-PowerBI.git
```

Move into the project directory:

```bash
cd Customer-Sales-Analytics-Using-Python-and-PowerBI
```

Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

---

## ▶️ How to Run

1. Open the Jupyter Notebook:

```bash
jupyter notebook
```

2. Open:

```
PROJECT_DHV.ipynb
```

3. Run all notebook cells.

4. Open the Power BI dashboard:

```
DHV PJT.pbix
```
